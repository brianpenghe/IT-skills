# Skill: Restore Mouse Right-Click by Reconnecting USB Receiver

## Problem

The mouse right-click suddenly stops working, sometimes only in certain
applications, while the rest of the mouse may continue to function
normally.

## Quick Fix

1.  Unplug the USB mouse receiver (or USB adapter/cable connected to the
    mouse).
2.  Wait 2--5 seconds.
3.  Plug it back into the USB port.
4.  Test the right-click again.

## Why This Works

Reconnecting the USB receiver forces the operating system to
re-enumerate the USB/HID device and reload its connection, which can
clear temporary communication or driver glitches.

## When to Try This

-   Right-click stops working.
-   Problem occurs across multiple applications.
-   Mouse otherwise appears connected.
-   Rebooting the application is undesirable.

## If It Doesn't Help

-   Test the mouse in another application.
-   Restart the affected application.
-   Restart Windows Explorer or reboot the computer.
-   Update or reinstall the mouse driver if the issue becomes frequent.
